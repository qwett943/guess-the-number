# guess-the-number
Простая игра "Угадай число" на Python. Компьютер загадывает число от X (Вашего первого числа) до Y (Ваше второе число), игрок угадывает с подсказками "больше/меньше".

## Как играть

1. Запусти программу
2. Выбери режим (1 или 2)
3. 1 режим: от 1 до 100
4. 2 режим: Свои числа x и y
5. Получай подсказки: «Больше» или «Меньше»
6. Повторяй, пока не угадаешь
7. Если угадать с первой попытки, комьютер поздравит!

## Технологии

- Python 3
