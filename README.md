# guess-the-number
Простая игра "Угадай число" на Python. Компьютер загадывает число от 1 до 100, игрок угадывает с подсказками "больше/меньше".

## Как играть

1. Запусти программу
2. Введи число
3. Получай подсказки: «Больше» или «Меньше»
4. Повторяй, пока не угадаешь

## Технологии

- Python 3

## Что я изучил, делая этот проект

- Генерация случайных чисел (`random.randint`)
- Цикл `while True` и `break`
- Условные операторы `if-elif-else`
- Ввод и преобразование типов
- Счётчик попыток
