# guess-the-number
Простая игра "Угадай число" на Python. Компьютер загадывает число от X (Вашего первого числа) до Y (Ваше второе число), игрок угадывает с подсказками "больше/меньше".

## Как играть

1. Запусти программу
2. Введи число X (От какого числа)
3. Введи число Y (До какого числа)
4. Получай подсказки: «Больше» или «Меньше»
5. Повторяй, пока не угадаешь
6. Если угадать с первой попытки, комьютер поздравит!

## Технологии

- Python 3
